# Портфолио студентки-разработчицы

## Описание проекта
Личный сайт-портфолио, созданный в рамках дисциплины "Фронтенд и бэкенд разработка".

## Технологии
- HTML5
- CSS3 с CSS-переменными
- Bootstrap 5
- Адаптивный дизайн
- Доступность (WCAG, ARIA)

## Особенности
Полностью адаптивный дизайн  
Семантическая HTML-разметка  
Доступность для скринридеров  
Адаптивные изображения  
Плавные анимации с учетом prefers-reduced-motion  

## Структура проекта
- `index.html` - главная страница
- `pages/` - дополнительные страницы
- `styles/` - CSS-стили
- `images/` - изображения
- `scripts/` - JavaScript файлы
