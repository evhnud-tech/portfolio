Запись 1: Начало пути в веб-разработке

01 января 2025

Тема: Основы HTML и CSS. Создание первой веб-страницы.

Изученные концепции:

  • Структура HTML-документа и базовые теги
  • Семантическая разметка (header, main, section, footer)
  • Основы CSS: селекторы, свойства, каскадность
  • Блочная модель и позиционирование

Практическая работа: Создала простую HTML-страницу с базовой структурой и стилизацией. Разобралась с подключением внешних CSS-файлов.

Рефлексия: Первые шаги в веб-разработке оказались увлекательными! Особенно понравилось сразу видеть результат своей работы в браузере.

Рефлексия: Bootstrap значительно ускоряет процесс разработки! Поражает, как быстро можно создавать профессионально выглядящие интерфейсы.

30 января 2025

Тема: Введение в JavaScript: переменные, функции и работа с DOM.

Изученные концепции:

  • Переменные (let, const) и типы данных

Рефлексия: JavaScript открыл новые горизонты! Теперь мои страницы действительно взаимодействуют с пользователем, а не просто статично отображают информацию.

Запись 4: Доступность (A11y) и веб-стандарты

10 февраля 2025

Тема: Веб-доступность (WCAG) и семантическая разметка.

Изученные концепции:

  • Принципы доступности WCAG 2.1
  • ARIA-атрибуты и их применение
  • Навигация с клавиатуры и скринридеры
  • Цветовой контраст и визуальная доступность
  • Семантические HTML5 теги

Практическая работа: Провела аудит доступности своего портфолио. Добавила skip-links, ARIA-атрибуты, улучшила семантику и цветовой контраст.

Рефлексия: Доступность - это не дополнительная функция, а неотъемлемая часть качественной веб-разработки. Важно создавать сайты для всех пользователей.

Запись 5: Современный CSS и производительность

20 февраля 2025

Тема: Современные возможности CSS и оптимизация производительности.

Изученные концепции:

  • CSS Grid и Flexbox для сложных раскладок
  • CSS-переменные (Custom Properties)
  • Контейнерные запросы (Container Queries)
  • Оптимизация изображений (WebP, AVIF)
  • Метрики производительности (LCP, CLS, FID)

Практическая работа: Реализовала адаптивную сетку проектов с использованием CSS Grid и контейнерных запросов. Оптимизировала изображения и добавила современные форматы.

Рефлексия: Современный CSS предоставляет мощные инструменты для создания гибких и производительных интерфейсов. Особенно впечатлили контейнерные запросы!
